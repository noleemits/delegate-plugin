PostScript Delegates plugin

Contributors: Astralis, Tech Life 


== Description ==
This plugin integrates the ability to manage delegates and member for PortScript, integrated with WooCommerce and AutomateWoo.

Features
- User registration with extra fields: first name, last name, phone, address, state, and city
- Ability to include up to 2 delegates in user registration
- Send registration invite to delegates (custom AutomateWoo trigger)
- Send reminder email to include delegated (custom AutomateWoo trigger)
- Dashboard tab for delegates management
- Ability to add/edit/delete delegates from Dashboard
- Dashboard tab for members management
- Ability to view members order's details
- Ability to view and edit logged in user extra fields in "Account details" tab


== Installation ==
1. Upload plugin zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set opt-out in AutomateWoo settings
4. Set up "Reminder to include delegates" AutomateWoo workflow. Trigger immediately.
5. Set up "Delegate invite" AutomateWoo workflow. Trigger 1 time per month.
